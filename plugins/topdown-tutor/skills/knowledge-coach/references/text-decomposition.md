# Text Decomposition

Use this when the topic has canonical or foundational texts.

## Goal

Turn original texts into teachable units without losing the value of reading the original material.

## For Each Original Text

Capture:

1. Historical context
2. Core claim
3. Plain-language meaning
4. Important terms:
   - original term
   - Chinese translation
   - plain Chinese explanation
5. Why it matters in the wider system
6. Common misunderstandings

## Output Pattern

For each selected text, provide:

- title
- why this text was selected
- what problem it is addressing
- the main claim in simple language
- how it connects to the overview
- what to pay attention to when reading it
