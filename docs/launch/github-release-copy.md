# GitHub Release Copy

## One-Line Pitch

TopDown Tutor helps agents teach hard topics top-down, gather the right materials, and verify real understanding.

## Feature Bullets

- Plain-language explanation first, then deeper layers
- Source gathering built into the teaching workflow
- Overview + original text + decomposition for canonical-text topics

## Install Snippet

```bash
gh skill install televgtamfe-coder/topdown-tutor knowledge-coach --agent codex --scope user
```

## Usage Snippet

```text
Teach me what an API is from the top down. Gather the right materials and check whether I really understood it.
```
