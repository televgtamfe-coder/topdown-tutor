# Brand Matrix

## Naming Roles

| Layer | Name | Purpose |
|---|---|---|
| Public brand (EN) | TopDown Tutor | GitHub, English README, launch posts |
| Public brand (ZH) | 从0到通 | Chinese README, Chinese content distribution |
| Repository | topdown-tutor | Public repo slug |
| Skill slug | knowledge-coach | Stable skill package name for v1 |
| Plugin name | topdown-tutor | Codex plugin package name |

## Positioning

- Top-down learning for hard topics
- Plain-language explanation first, then deeper layers
- Material acquisition included
- Original text plus decomposition supported
- Understanding checks built in

## Naming Decision

- Do not rename the skill slug in v1.
- Use the public brand in repo docs and launch assets.
- Revisit skill-slug rename only after the public package is stable.
