# Demo Script

## Demo Goal

Show that TopDown Tutor can explain a hard topic clearly, gather materials, preserve source quality, and check whether the learner really understood it.

## Demo Topic Order

1. API
2. React Hooks
3. Probability
4. The Art of War

## Talking Points

1. Start from the overall map instead of diving straight into details.
2. Keep explanation simple first without downgrading source authority.
3. Support canonical-text learning with overview + original text + decomposition.
4. Translate technical or English terms on first mention.
5. Ask up to three key checks instead of scattering many questions.
