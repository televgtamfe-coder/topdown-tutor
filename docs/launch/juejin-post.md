# 掘金发布草稿

## 标题

我做了一个让 Agent 从上到下讲懂知识、顺手找材料、还能检查是否真懂的学习 Skill

## 痛点

很多“讲解型”提示词只有解释，没有真正教学流程。更常见的问题是，学习者还要自己去找材料，结果材料质量和讲解逻辑经常脱节。

## 方法

从0到通把三件事绑在一起：

1. 固定讲解节奏
2. 主题对应的材料获取
3. 少量高价值理解检查

## 示例

它可以讲 API、React Hooks、概率，也可以按“综述 + 原典 + 拆解”方式处理《孙子兵法》这类原典主题。

## 安装入口

```bash
gh skill install televgtamfe-coder/topdown-tutor knowledge-coach --agent codex --scope user
```

## 行动引导

如果你也在做学习型 Agent、知识型工作流或者教学型提示工程，可以直接拿这个结构去改。
