# Starting Point Breakdown

Use this when the learner has no clear starting point or cannot tell where the topic begins.

## Goal

Create a short orientation layer before the normal lesson so the learner knows what the topic is, why it matters, and where to begin.

## Output Pattern

Cover these in order:

1. What this topic is
2. Why it matters
3. Where it sits in the larger map
4. The three core pieces to notice first
5. The first important technical term or English term, if any:
   - original term
   - Chinese translation
   - plain Chinese explanation
6. One easy example
7. The next step after this overview
